# Thrive 55+

Thrive 55+ Nursing Advantage™ is a program that helps experienced nurses aged 55 and older find career clarity, reset emotionally, and navigate professional transitions — built around **The Thrive 55+ Career Direction Method™** ("Name the strain. Capture your value. Investigate your options.").

This repository holds the project's planning documents, course content, brand assets, and design prototypes, and it is where the **AI-powered learning platform** described in [`PLAN.md`](PLAN.md) will be built.

## The Project

Two collaborators are creating online courses for nurses 55+:

- **Melissa Jo Tully** — owns this repository and the technology (hosting, database, AI accounts); leads the platform build and outcomes research.
- **Sue Adair** — on-camera instructor and content author; her Wix site sells the course and links students into the platform.

The first course teaches nurses 55+ to use artificial intelligence with confidence and without fear. The platform (see `PLAN.md`) delivers four modules in sequence: an AI tutor chat, a survey/outcomes engine, a student portal, and voice read-aloud.
